A Channel Set – Sets a specific set of channels to operate on in the 5 GHz band. Valid operating channels are constrained by the configured regulatory domain. Auth Server Type – Indicates the type of authentication server being used for EAP. Auto Profile – Manages the auto profile mode. BG Channel Set – Sets a specific set of channels to operate on in the 2.4 GHz band. Valid operating channels are constrained by the configured regulatory domain. Beacon Miss Time – Indicates the beacon missed time in TUs before considering to switch to a different BSS. BT Coexist – Enables or disables Bluetooth coexistence. CCX features – Enables or disables CCX features. Certificate Path – Used to change the path of the certificate store. Date Check – Validates certificates against the system date and time. Default Adhoc Channel – Indicates the channel to be used for an ad hoc connection if the active profile has a Radio Mode value of Ad Hoc. DFS Channel – Indicates support (or lack of support) for 5 GHz (802.11a) channels where DFS is required. Frag Threshold – If packet size (in bytes) exceeds this threshold, then the packet is fragmented. FIPS – Turns FIPS compatibility on (Supporting FIPS requires additional integration efforts). PMK Caching – Indicates the type of PMK caching to use with a WPA2 encryption type. Probe Delay – Maximum number of seconds to wait between scans during a connection attempt. Regulatory Domain – Shows the current regulatory domain. Roam Period ms – The amount of time between roam scans. Roam scans occur after the radio has fallen below the roam trigger. Roam Trigger – The signal strength (RSSI) (in dBm) at which the radio scans for an access point with a better signal strength. RTS Threshold – The packet size above which RST/CTS is required on link. Scan DFS Time – Indicates the dwell (listen) time when passively scanning on a DFS channel. TTLS Inner Method – Indicates the authentication method that is used within the secure tunnel created by EAP-TTLS. UAPSD – Enables or disables UAPSD features. WMM – Enables or disables WMM features.
